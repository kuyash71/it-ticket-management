# Phase Roadmap

## MVP (Bu Faz)

Odak:
- Ticket lifecycle
- Priority + SLA yönetimi
- RBAC
- Audit/Timeline görünürlük
- Attachment ve complaint temelleri

Kısıt:
- Event-driven projection zorunlu değil
- Gelişmiş otomasyon yok
- Harici entegrasyon yok

## Phase-2

Planlanan genişleme:
- Event granularity ayrıştırma
- Çoklu SLA clock desteği
- Notification motoru genişletme
- CQRS değerlendirmesi
- Reporting projection/read-model geçişi

## Phase Geçiş Kriterleri

- MVP acceptance kriterleri tam karşılanmalı
- CI pipeline stabil olmalı
- Kritik akışlarda operasyonel geribildirim toplanmış olmalı
